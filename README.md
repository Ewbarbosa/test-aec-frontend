## Instruções

Primeiro, clone o repositório.

Em seguida execute:

```bash
npm install
#ou
yarn install
```

Após ter baixado as dependências, execute:

```bash
yarn dev
#ou
npm run dev
```

Feito isso, a aplicação irá começar a escutar na porta 3000 (porta padrão).

E por fim, acesse: 
http://localhost:3000
